#  Блог на минималках : Yii2

## Установка
* Создать любой host. Например: blog.local
* Создать базу данных blog2
* Импортировать файл базы данны blog2.sql
* Или в корневой директории выполнить консольную команду
	php yii migrate
запустив миграции
